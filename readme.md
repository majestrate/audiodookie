# audio dookie

wayland compositor wallpaper that is a mpd / pulseaudio audio visualizer

works with [sway](https://github.com/swaywm/sway) (probably)

## building

### deps

* cmake
* c++ 17 compiler
* wayland 
* fftw
* pulseaudio
* cairo

### build process

standard cmake build process

    $ mkdir -p build && cmake build && make
